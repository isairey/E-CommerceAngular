# 🌌 Acerca del proyecto

**Online Store Angular App** es una plataforma web de comercio electrónico creada con **Angular**, enfocada en la gestión y visualización de productos tecnológicos, moda, accesorios y compras en línea.

La aplicación permite:

- 🛒 Mostrar productos dinámicamente
- 📦 Gestionar inventario
- 🔎 Buscar productos
- 👤 Administrar usuarios
- ❤️ Gestionar favoritos
- 🛍️ Controlar carrito de compras
- 💳 Simular proceso de pago
- 📱 Navegación moderna SPA

El proyecto utiliza una arquitectura modular y escalable basada en Angular CLI.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/E-CommerceAngular.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd E-CommerceAngular
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar aplicación

```bash
ng serve
```

Abrir en navegador:

```bash
http://localhost:4200/
```
